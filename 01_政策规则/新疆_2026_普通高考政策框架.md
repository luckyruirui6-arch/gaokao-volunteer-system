# 科目与分数理解

- 科目结构：语文、数学、外语 + 文综/理综或当地对应科目结构，具体以当地 2026 年招生工作规定为准。
- 分数口径：一般按传统文理分科总分口径理解，分数线和位次解释应优先参考本地官方数据。
- 填报思路：先看政策和计划，再看普通批志愿结构，不要直接照搬新高考省份的比较方法。

# 志愿填报时最该先确认什么

1. 当年是否仍沿用当前考试模式。
2. 本科普通批的投档单元是 院校、专业（类） 还是 院校专业组。
3. 本科普通批和提前批的志愿个数、是否平行志愿、是否允许专业调剂。
4. 特殊类型招生、专项计划、定向培养、公费项目是否单列批次或放在提前批。
5. 本省 2026 年一分一段、控制线、投档线公布节奏。

# 本省填报提醒

新疆当前更要重点看招生计划、政策要求和区域机会，普通批分数比较不能脱离本地口径。
