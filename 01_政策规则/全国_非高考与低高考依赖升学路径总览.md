# 先把概念说清楚

家长常说的“`不用高考也能上大学`”并不都是一个概念。实际可以分成：

- 完全不走普通高考统招主路径
- 仍需高考，但不是只靠高考分
- 仍需高考，只是多了一条特殊招生通道

# 相对接近“不用普通高考统招”的路径

- 保送生
- 运动训练、武术与民族传统体育专业单招
- 高职单招
- 部分港澳高校独立招生

# 仍需高考，但不是只靠高考分

- 强基计划
- 综合评价招生
- 高水平运动队
- 艺术类招生
- 高校专项计划

# 最容易被误解的点

- 强基计划不是免高考
- 自强计划/高校专项不是免高考
- 艺术类不是免高考
- 高水平运动队不是免高考
- 港澳高校也不是都不看高考
