# 科目与分数理解

- 科目结构：语文、数学、外语 3 门统一高考科目 + 首选科目（物理/历史）1 门 + 再选科目 2 门。
- 分数口径：首选科目通常按原始分计入总分，再选科目一般按等级赋分计入总分，具体以省考试院说明为准。
- 填报思路：先判断首选科目，再看专业组或专业类，最后才看学校最低分。

# 志愿填报时最该先确认什么

1. 当年是否仍沿用当前考试模式。
2. 本科普通批的投档单元是 院校、专业（类） 还是 院校专业组。
3. 本科普通批和提前批的志愿个数、是否平行志愿、是否允许专业调剂。
4. 特殊类型招生、专项计划、定向培养、公费项目是否单列批次或放在提前批。
5. 本省 2026 年一分一段、控制线、投档线公布节奏。

# 本省填报提醒

江西新高考转型期最怕用老思路填新志愿，尤其是首选科目不同导致的比较失真。
