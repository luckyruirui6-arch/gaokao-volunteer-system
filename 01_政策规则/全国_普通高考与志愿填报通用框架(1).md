# 核心结论

全国普通高考的底层框架是统一高考报名、考试、体检、志愿、投档、录取，但 `批次设置`、`志愿数量`、`是否院校专业组`、`是否新高考`、`加分和照顾政策落地方式` 由各省级招生考试机构具体公布。
截至 `2026-04-10`，仓库尚未统一落地各省 `2026` 正式工作规定，因此本页主要提供全国通用框架和阅读省级政策时的抓手。

# 读省级规则时必须抓的字段

- 考试模式：`3+3`、`3+1+2`、传统文理科或其他省级模式
- 批次设置：本科提前批、本科普通批、专科批等
- 志愿模式：平行志愿、顺序志愿、院校专业组、专业+院校
- 选科要求：尤其是新高考省份
- 调剂规则：是院校内调剂、专业组内调剂，还是其他口径
- 提前批包含什么：军校、公安、司法院校、公费师范、免费医学生、优师专项、定向招生等
- 特殊类型位置：强基计划、高校专项、高水平运动队、综合评价、保送生等在哪个时间节点和批次衔接
- 加分和照顾政策：是否保留、如何使用、谁能享受

# 家长最容易踩坑的通用误区

- 全国高考规则并不一样
- 控制线不等于稳录线
- 特招和提前批会直接影响志愿机会
- 新高考省份往往不能只看学校名
