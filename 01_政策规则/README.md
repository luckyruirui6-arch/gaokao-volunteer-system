# 使用顺序

1. 先看全国通用框架，明确普通批、提前批、特殊类型招生的边界。
2. 再看省级政策工作版，理解本省考试模式和当年仍需确认的变量。
3. 最后回到 02_省份数据 和 03_院校库 进行分数、位次、学校层面的判断。

# 提醒

- 各省 2026 正式招生工作规定发布时间不同。
- 本目录中的省级文件先作为 工作版 使用，等正式文件落仓后再覆盖更新。
