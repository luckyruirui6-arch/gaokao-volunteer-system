# 文件定位

- `全国公办本科院校总表_原始.csv`
  用户提供的全国院校原始底表。
- `全国公办本科院校总表_清洗.csv`
  清洗后的全国院校表，适合继续加工。
- `全国公办本科院校总表_严格本科去重.csv`
  适合做全国范围统一口径统计和后续数据处理。
- `全国公办本科院校总表_严格本科去重.md`
  适合在 Obsidian 内快速查看全国层级底表。
- `全国公办本科院校总表.md`
  全国院校知识总表，适合知识库入口使用。
- `全国所有公办本科院校总表.md`
  全国层面的说明页 / 总入口页。

# 使用建议

1. 先看 `全国公办本科院校总表.md` 或 `全国公办本科院校总表_严格本科去重.md`，快速了解全国底表结构。
2. 再进入 `分省总表`，定位到具体省份。
3. 最后回到 `03_院校库/省份` 看分省总表和单校页。
4. 如果后面要给 AnythingLLM 用，优先用清洗后的 Markdown 总表，不要直接把原始 CSV 当最终知识文档。

# 主入口规则

- 全国层面的学校入口优先使用清洗后的 Markdown 总表，不直接把原始 CSV 当主入口。
- 学校名统一以全角括号版本为主入口；如果历史文件里存在半角括号别名页，只作为兼容入口，不再当作索引主入口。
