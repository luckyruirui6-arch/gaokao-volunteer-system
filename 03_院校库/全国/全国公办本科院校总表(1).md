# 字段说明

- `学校标识码 / 主管部门 / 所在地 / 办学层次 / 备注`：来自教育部 2025 全国普通高等学校名单
- `985 / 211 / 双一流`：当前用于知识库整理的标签字段
- `隶属关系`：按主管部门归纳为教育部直属、中央部委所属、地方所属等
- `院校层级`：按 985 / 211 / 双一流 标签自动归纳

# 使用提醒

- 这份表适合做全国院校底表、筛选器和知识库索引
- `双一流` 字段当前按第二轮建设名单口径做整理，后续仍建议结合正式官方名单继续抽查校对
- AnythingLLM 使用时，建议优先导入清洗版 CSV 或再导出一份 Markdown 分省总表
