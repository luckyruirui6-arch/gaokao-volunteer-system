# 学校定位

教育部直属。华南第一学府。以全学科均衡发展著称。

# 优势专业

- **[[临床医学]]/工商管理**：全国排名极其靠前，在大湾区认可度无敌。

# 适合什么学生

- **位次段**：该省顶尖生源，物理类考生建议关注选科“物化绑定”要求。

# 就业与升学

- **深造质量**：具有极高的保研率和名企录用优先权。
