# 学校定位

广东省/深圳市共建。地处腾讯隔壁。被誉为最快进步大学。

# 优势专业

- **计算机/信息工程**：得益于深圳华为腾讯等企业的地缘优势，其实战就业力极大。

# 适合什么学生

- **位次段**：该省顶尖生源，物理类考生建议关注选科“物化绑定”要求。

# 就业与升学

- **深造质量**：具有极高的保研率和名企录用优先权。
