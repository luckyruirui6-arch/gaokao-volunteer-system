# 学校定位

四川省属。以“川农大精神”著称。在水稻育种、畜牧方向具有极强实力。

# 优势专业

- **特色学科**：在该领域具有全国及行业领先地位。

# 适合什么学生

- **位次段**：该省顶尖生源，物理类考生建议关注选科“物化绑定”要求。

# 就业与升学

- **深造质量**：具有极高的保研率和名企录用优先权。
