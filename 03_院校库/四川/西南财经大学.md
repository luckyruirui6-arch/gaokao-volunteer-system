# 学校定位

教育部直属。西南地区金融人才的摇篮。适合追求高品质商科教育的精英考生。

# 优势专业

- **[[金融学]]/[[会计学]]**：在成渝金融中心具有主导性的行业地位。

# 适合什么学生

- **位次段**：该省顶尖生源，物理类考生建议关注选科“物化绑定”要求。

# 就业与升学

- **深造质量**：具有极高的保研率和名企录用优先权。
