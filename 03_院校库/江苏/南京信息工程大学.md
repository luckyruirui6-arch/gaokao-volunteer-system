# 学校定位

原南京气象学院，被誉为“气象人才的摇篮”。适合关注环保、防灾减灾及希望在特定细分领域占据垄断优势的考生。

# 优势专业

- **大气科学**：全国第一，基本上垄断了气象系统的招录机会。

# 适合什么学生

- **位次段**：省属头部院校，通常建议位次在前 40,000 名以内（部分专业要求更高）。

# 就业与升学

- **本地就业红利**：在江苏当地及长三角地区具有极高的雇主认可度。
