# 学校定位

工信部直属。国防七子之一。被誉为“兵器技术人才摇篮”。适合务实、追求军工背景及电子通信技术的考生。

# 优势专业

- **兵器科学/光学**：全国顶尖，掌握了火炮及光电对抗的核心技术。
- **计算机/[[自动化]]**：侧重于特种装备控制，就业质量极高。

# 适合什么学生

- **位次段**：211 级考生，物理类位次建议在前 15,000 名以内（不同学校差异大）。

# 就业与升学

- **行业认可度**：作为 211 高校，在长三角及全国头部企业招聘中均处于第一名单。

# 报考风险

- **专业分化风险**：江苏院校专业组设置较细，需注意冷门专业的调剂可能。

江苏的 211 没有弱校。选南大是选名分，选苏大是选生活，选南理工南航是选这辈子的技术饭碗。
