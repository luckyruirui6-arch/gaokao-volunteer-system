# 学校定位

江苏教育系统的标杆。拥有随园“东方最美校园”。适合热爱教育、追求深厚人文素养的考生。

# 优势专业

- **马克思主义理论/教育学/美术**：在全国具有极高的影响力。
- **[[法学]]**：在省内司法体系中拥有强大的校友力量。

# 适合什么学生

- **位次段**：211 级考生，物理类位次建议在前 15,000 名以内（不同学校差异大）。

# 就业与升学

- **行业认可度**：作为 211 高校，在长三角及全国头部企业招聘中均处于第一名单。
