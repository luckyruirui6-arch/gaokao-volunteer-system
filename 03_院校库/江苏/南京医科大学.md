# 学校定位

江苏省内最顶尖的医科院校，录取分数常年高居省属校榜首。

# 优势专业

- **[[临床医学]]/[[口腔医学]]**：附属医院资源极其庞大，在长三角医疗系统口碑无敌。

# 适合什么学生

- **位次段**：省属头部院校，通常建议位次在前 40,000 名以内（部分专业要求更高）。

# 就业与升学

- **本地就业红利**：在江苏当地及长三角地区具有极高的雇主认可度。
