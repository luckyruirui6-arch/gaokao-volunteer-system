# 学校定位

上海市属公办、由上海市政府与国家电网共建。它是中国电力院校“二龙四虎”架构之外的另一支地方劲旅，在动力工程及电力系统[[自动化]]方向具有显著优势。适合立志进入能源电力系统、追求高就业质量且务实的考生。

# 优势专业

- **[[电气工程及其自动化]]**：学校的“当家花旦”，毕业生在国家电网上海市电力公司极其受青睐。
- **[[能源与动力工程]]**：侧重火电及冷热联供技术，培养了大批一线的动力技术骨干。
- **[[自动化]]/测控**：结合智能电网发展方向，在工业控制领域就业面极广。

# 适合什么学生

- **位次段**：上海位次约在 16,000-24,000 名左右。
- **职业目标**：国家电网、发电集团、各级设计院、高端能源管理企业。
- **特质**：数理基础坚实、能吃苦、具备较强的工程实践敏锐度和规矩意识。

# 就业与升学

- **行业直通优势**：在上海电网招聘中，上电力毕业生的入职人数常年稳居前茅。 
- **稳定性卓越**：电力系统属于高壁垒且极具抗周期性的行业，职业发展稳健。

# 报考风险

- **单一行业依赖**：若不打算进电力系统，其在通用互联网金融领域的溢价需结合专业具体平衡。
- **招生名次攀升**：由于就业太好，近年来投档分持续看涨，捡漏概率变小。

# 张雪峰风格点评

如果你家孩子以后想进上海国家电网，除了清华和华电，上电力就是最实惠的选择。它是典型的“行业饭碗”校。在电力这个圈子里，你是自己人。别看校名没那么高大上，毕业后进个稳定的国企、拿一份体面的薪水，这学校能给你的是看得见摸得着的福报。买的是“确定性”。
