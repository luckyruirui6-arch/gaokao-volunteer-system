# 学校定位

上海市属公办本科院校。依托上海产业发展需求，在民航[[飞行技术]]、轨道交通、多媒体设计及服装营销领域具有显著优势。学校产学研结合效果优良，是上海现代服务业与制造业交叉学科的人才高地。适合关注交通科技、[[工业设计]]及希望进入航空铁路系统的考生。

# 优势专业

- **[[飞行技术]]/飞行器制造**：培养民航飞行员及航空工程人才，与国内外各大航空公司有深度订单培养协议。
- **轨道交通信号与控制**：针对城市地铁与动车组运维，具有较高的行业准入壁垒。
- **服装设计与工程**：侧重服装产业链经营，与东华大学形成良好的错位竞争与补充。

# 适合什么学生

- **位次段**：上海位次约在 22,000-32,000 名左右。
- **职业路径**：大型航空公司、城市地铁运营公司、大型制造中心研发岗、时尚品牌营销专家。
- **特质**：纪律性好（飞行等专业要求高）、具备一定的艺术审美（设计类）、工程协作能力强。

# 就业与升学

- **行业渗透力强**：在上海的轨道交通和民航系统中，工程大的校友圈层非常活跃。
- **综合就业质量**：由于地处松江大学城，资源共享度高，综合社会化就业竞争力不俗。

# 报考风险

- **特殊专业体检**：飞行等专业对视力、身高有极其严格的国家硬指标要求，报考需先行参检。
- **跨校区管理**：多校区布局需注意具体的学院落脚点。

# 张雪峰风格点评

工程大是上海的“通才工厂”。如果你想当机长、想进地铁当总监，或者想搞服装生意，工程大都能给你最实在的资源。它比普通的大学多了一份“行业背景”。别小看它，在上海这种特别讲究行业背景的地方，工程大的招牌是很好使的。它现在的航空大楼和先进实验室，那是非常带感的。选这儿，就是为了那个特定的圈子。
