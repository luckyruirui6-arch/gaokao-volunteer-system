# 学校定位

中央直属公办本科院校，国家“双一流”建设高校。华东师大是中国唯二的 985 师范顶尖校（另一所为北师大），在教育研究、[[心理学]]及人文、地理学领域优势显著。适合立志于高端教育事业或深厚人文研究的考生。

# 优势专业

- **教育学/[[心理学]]**：学科评估 A+，在基础教育改革及心理咨询方向具有行业话语权。
- **中文/历史/数学/物理**：基础学科强劲，主要培养上海及长三角各区名校的高端师资。
- **地理学**：在河口海岸及城市生态方向国内领先。

# 适合什么学生

- **位次段**：上海位次约在 5,000-8,000 名左右。
- **职业目标**：特大城市三甲高中/大学教师、教育行政部门、知名出版社、心理咨询专家。
- **特质**：沟通亲和力强、性格温润稳重、具备较强的学术自驱力。

# 就业与升学

- **上海教育圈首选**：在上海的中小学系统，华东师大毕业生的认可度和入职率极高。
- **多元化发展**：作为 985 高校，不仅限于当老师，在政府机关和大型企业的职能部门也有较强表现。

# 报考风险

- **师范编制竞争**：随着高端人才涌入，师范生在热门地区的入职难度和学历要求在持续升高。
- **校区位置分化**：闵行校区较偏，中环北校区（普陀）虽然便利但教学重心已大幅外迁。
