# 学校定位

上海市属公办本科院校。由原上海立信[[会计学]]院和原上海[[金融学]]院合并组建。被誉为“中国现代会计教育的发祥地”，在上海财务及金融实务界具有极强的影响力和极其庞大的校友基础。适合追求实用技能、立志于在上海从事财务金融审计工作的考生。

# 优势专业

- **[[会计学]]**：学校的“灵魂”专业，教材与实务紧密挂轨，在长三角地区的“四大”及内资事务所录用榜单中高居前列。
- **[[金融学]]/[[审计学]]**：依托上海金融中心的资源，在银行、保险及税务审计方向具有极强的行业应用背景。
- **[[资产评估]]**：侧重实务操作，专业壁垒高，就业对口率优秀。

# 适合什么学生

- **位次段**：上海位次约在 18,000-26,000 名左右。
- **职业目标**：大型国企财务部、各类银行网点/中台、会计师事务所审计专员、税务代办机构。
- **特质**：做事严谨、对数字敏感、能吃苦（审计行业强度大）、具备较强的考证毅力（CPA/ACCA）。

# 就业与升学

- **高对口率**：毕业生在上海财务市场的占有率极高，很多中小企业的财务主管及甚至上市公司 CFO 都有立信背景。
- **实习资源**：由于校友遍布，学生在校期间获取高质量财务实习机会的能力显著优于同档次综合校。

# 报考风险

- **行业红利期波动**：随着 AI [[自动化]]入账技术普及，基础会计岗位需求萎缩，必须走高端管理会计路线。
- **校区差异**：浦东与松江校区运行，部分学生需适应跨校区的过渡。

# 张雪峰风格点评

立信在上海就是“好就业”的代名词。如果你家孩子分儿够不上财大，但又想在上海滩端稳财务这碗饭，立信就是最实在的选择。它的文凭在上海财务部经理的眼里，比外地很多985都好使，因为你是“立信”出来的，代表了专业和靠谱。这是一所能让你靠本领扎根上海的学校。
