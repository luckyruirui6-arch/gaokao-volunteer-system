# 学校定位

非211中最硬核的工科学校之一。化工、机械在长三角认可度极高。

# 适合什么学生

- **分数段**：全省前3万名左右
- **职业目标**：现实就业派，目标长三角工矿/制药企业岗位。
- **决策建议**：化工是王牌，但在新高考下要看准选科建议。

# 张雪峰风格点评

长三角‘小霸王’。虽然没进211，但它的化工在行业内也是有头有脸的。如果你分数上不了浙大，想在杭州找份体面工科工作，选它很稳。
