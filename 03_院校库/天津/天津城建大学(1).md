# 学校定位

天津市属公办。为城市建设全产业链提供技术支持。适合追求细分工程行业准入资质的考生。

# 优势专业

- **[[土木工程]]/水务工程**：针对北方寒冷地区与沿海环境的城建方案极其成熟。

# 适合什么学生

- **特质**：务实稳健，追求特定行业就业保障。

# 就业与升学

- **主要流向**：本市及周边地区相关行业国企、校办企业及事业单位。

# 报考风险

- **行业局限性**：专业针对性极强，跨行择业需付出额外努力。

# 张雪峰风格点评

城建大是“基建后备军”。如果你想在这个行业里混个出身，城建大的牌子在北方各大设计院还是数得上的。买的是“行业配套感”。
