# 学校定位

天津市属公办本科院校。旨在培养具有国际视野的应用型工程技术人才。适合追求硬核理工背景、且关注天津高新制造业的考生。

# 优势专业

- **[[计算机科学与技术]]**：侧重应用软件开发与[[网络安全]]，在天津 IT 市场有良好口碑。
- **[[环境工程]]**：结合天津工业水处理特色，办学沉淀厚。

# 适合什么学生

- **位次段**：在天津位次约在 22,000-30,000 名左右。
- **职业目标**：各类高新科技企业研发员、政府环境监测岗。

# 就业与升学

- **主要流向**：本市及周边地区相关行业国企、校办企业及事业单位。

# 张雪峰风格点评

天理工是一所“稳扎稳打”的学校。它虽然名中带理工，但它的[[工业设计]]和社工类也挺有特色。选它，买的是一份“务实”。
