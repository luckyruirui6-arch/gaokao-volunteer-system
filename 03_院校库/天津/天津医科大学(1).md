# 学校定位

天津市属公办本科院校。被誉为新中国第一所医学院校。天医大在肿瘤学、基础医学及心血管疾病领域具有顶尖学术实力。适合位次较高、具备仁爱之心且希望在北方医疗系统深造的考生。

# 优势专业

- **肿瘤学**：依托附属肿瘤医院，在该领域居全国极领先地位。
- **[[临床医学]]/[[口腔医学]]**：教学严谨，毕业生在北京、天津各大三甲医院认可度极高。
- **基础医学**：拥有强大的硕博点支撑，适合有志于医学科研的尖子生。

# 适合什么学生

- **位次段**：在天津位次约在 5,000-8,000 名内。
- **职业目标**：三甲医院主治医师、医学科研专家、公共卫生管理人员。
- **特质**：性格沉稳细致、记忆力超群、抗压能力强。

# 就业与升学

- **高进阶就业**：由于是 211 院校，其本硕一体化及保研外校的底气非常足。
- **京津冀联动**：在华北地区的医疗网络中，天医校友分布极广。

# 报考风险

- **培养周期长**：医学必须做读研乃至读博的十年长跑心理准备。
- **校区位置较窄**：主校区位处市中心核心区，校园规模相对有限。
