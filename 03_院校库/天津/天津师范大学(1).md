# 学校定位

天津市属公办本科院校。天津基础教育的定海神针。适合热爱教学事业、追求稳定职业且希望留津发展的考生。

# 优势专业

- **小学/[[学前教育]]**：天津公办教师编制的最大源头之一。
- **[[心理学]]/政治学**：在全国学科评估中具有显著优势，具备较强的学术后劲。

# 适合什么学生

- **位次段**：在天津位次约在 18,000-25,000 名左右。
- **职业目标**：天津各区名校教师、教育行政部门、知名出版社。

# 就业与升学

- **系统影响力**：在天津教育系统内，天师大的校友遍布各大中小学校长层。
