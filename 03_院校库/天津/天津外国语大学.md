# 学校定位

天津市属公办本科院校。北方特别是环渤海地区重要的多语种人才库。适合具备语言感知力、立志国际化职业路径的考生。

# 优势专业

- **多语种集群**：[[英语]]、[[日语]]、[[俄语]]均有深厚的经贸应用特色。
- **国际商务**：侧重语言与商科的结合。

# 适合什么学生

- **位次段**：在天津位次约在 20,000-30,000 名左右。

# 就业与升学

- **主要流向**：本市及周边地区相关行业国企、校办企业及事业单位。

- **行业局限性**：专业针对性极强，跨行择业需付出额外努力。

# 张雪峰风格点评

天外是“天津的窗口”。这学校在天津人心目中地位不低。如果你外语好、又不排斥社交，选择它去搞外企或者外展是很明智的。买的是那份“洋气”。
