# 学校定位

天津市属公办。被誉为“中国职教师资的摇篮”。它是全国第一家专门培养职业教育老师的本科院校。适合想当“工程师中的老师”或“老师中的工程师”的考生。

# 优势专业

- **机械/电气（职教方向）**：毕业生在全国大中专职业学校、技师学院极其抢手。

# 适合什么学生

- **特质**：务实稳健，追求特定行业就业保障。

# 就业与升学

- **主要流向**：本市及周边地区相关行业国企、校办企业及事业单位。

# 报考风险

- **行业局限性**：专业针对性极强，跨行择业需付出额外努力。

# 张雪峰风格点评

这学校是“避雷神校”。大家都在卷重点中学老师，这儿的学生在抢全国职业学校的编制。就业率高得吓人。如果你想稳稳当当地当个带编制的老师，分儿又不是特别顶尖，这儿就是金饭碗。买的是那份“错位竞争”。
