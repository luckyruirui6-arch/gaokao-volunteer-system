# 学校定位

教育部直属公办本科院校，国家“双一流”建设高校。前身为北洋大学，是中国第一所现代大学。天大是中国工科实力最强的大学之一，以“实事求是”为校训。适合务实、硬核且追求顶尖工程技术的理科学霸。

# 优势专业

- **化学工程与技术**：全国多年排名第一，代表了中国在该领域的最高学术成就。
- **测量/建筑/仪器**：精密仪器、光学工程及[[建筑学]]（建筑老八校）在业内具有垄断级地位。
- **新工科集群**：在储能、[[人工智能]]、海洋工程等前沿领域布局早、资源多。

# 适合什么学生

- **位次段**：天津位次通常在 2,500-3,500 名左右。
- **职业路径**：大型工业研发大院、顶级制造企业、知名建筑设计院、高端芯片及新材料公司。
- **特质**：务实踏实、逻辑严密、具备极强的解决复杂工程问题的意愿和能力。

# 就业与升学

- **工业界认可度**：在全国大型制造、能源及基建央企中，天大毕业生的认可度极高。
- **薪资表现**：由于专业高度契合国家战略需求，毕业生起薪处于全国高校前列。

# 报考风险

- **学习负荷重**：课程难度大，特别是化工及机械大类，必须做好“死磕”专业的准备。
- **男女比例**：典型的理工校环境，校园文化相对硬朗简洁。

# 张雪峰风格点评

天大就是“实干派”的典范。如果你想搞硬核科技、想实打实地解决问题，来天大准没错。它在上海和长三角的工业界非常吃香。我提醒想学建筑的孩子：天大建筑是国内绝对的梯队，分极高。选天大，就是选了一个极其稳固、极其专业的技术人生。
