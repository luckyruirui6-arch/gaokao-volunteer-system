# 学校定位

位于革命圣地延安。作为老牌综合性大学，其医学（原延大医学院）和化工在全省具有极强竞争力。

# 适合什么学生

- **分数段**：全省前5万名（物理）左右
- **职业目标**：认可延大品牌，目标医学执照或能源化工系统。
- **决策建议**：医学和化工是延大的两条硬腿，就业非常稳健。

# 张雪峰风格点评

延大不只有革命传统，它的医学院那是真硬。在西北地区，延大的医生校友多得是。如果你分数够不上西医大，来延大读临床医学，毕业了回省城大医院也不丢脸。
