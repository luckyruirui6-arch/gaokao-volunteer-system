# 学校定位

位于咸阳市。以教师教育为核心，书法和学前教育是全省亮点。

# 适合什么学生

- **分数段**：全省前8万名左右
- **职业目标**：立志从事基础教育，特别是书法或幼教方向。
- **决策建议**：地理位置紧邻西安，就业资源可与西安共享。

# 张雪峰风格点评

咸阳师范就在西安边上，位置极佳。它的书法学在全国都有名气。如果你想以后在西安周边当老师，咸阳师范是个非常实惠的跳板。
