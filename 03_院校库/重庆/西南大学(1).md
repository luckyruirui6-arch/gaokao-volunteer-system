# 学校定位

教育部直属公办本科院校，国家“双一流”建设高校。由原西南师范大学与原西南农业大学合并组建。学校在[[心理学]]、教育学、生物学及农学领域具有国际级水平。适合性格温婉、热爱生命科学或立志教育事业的博学型考生。

# 优势专业

- **[[心理学]]**：全国学科评估连续 A+，代表了我国在情感心理与认知领域的最高研究水平。
- **教育学/文学**：部属师范属性加持，在西南基础教育系统具有绝对统治力。
- **蚕桑/农学**：拥有家蚕基因组生物学等全球顶尖实验室支撑。

# 适合什么学生

- **位次段**：重庆物理类位次约在 5,000-10,000 名左右。
- **职业目标**：顶尖公办学校教师、心理健康专家、高端科研机构研究员。
- **特质**：耐心细致、具备较强的人文情怀与沟通能力。

# 就业与升学

- **主要流向**：西南地区大中型国企、事业单位及知名民企。
