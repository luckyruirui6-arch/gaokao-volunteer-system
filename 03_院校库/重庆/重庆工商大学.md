# 学校定位

侧重商业管理与国际经贸，是西南地区重要的商科人才库。

# 优势专业

- **工商管理/会计**：实战能力强，毕业生在成渝地区的知名企业中非常活跃。

# 适合什么学生

- **特质**：务实肯干，追求特定行业职业门槛。

# 就业与升学

- **主要流向**：西南地区大中型国企、事业单位及知名民企。

- **专业绑定感强**：适合职业指向明确的考生，改行成本相对存在。

# 张雪峰风格点评

重庆工商是“魔幻 5A 景区校”。位置好、氛围活。如果你以后想在大城市做个白领、搞搞财务管理，这儿最合适。买的是一份“接地气”。
