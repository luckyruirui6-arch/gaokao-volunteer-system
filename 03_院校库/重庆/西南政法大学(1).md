# 学校定位

重庆市属公办本科院校。被誉为中国[[法学]]界的“黄埔军校”。作为“五院四系”的元老，西政在政法系统拥有极其庞大且互助性极强的校友网络。适合逻辑极强、立志于政法系统发展的硬核考生。

# 优势专业

- **[[法学]]**：绝对垄断性学科，培养了大量大法官、大检察官及顶尖律师。
- **刑事技术/[[新闻学]]**：侧重法治化应用，具有极佳实战特色。

# 适合什么学生

- **位次段**：重庆物理类位次约在 5,000-12,000 名内（[[法学]]专业组分极高）。
- **职业路径**：各级法院/检察院/公安机关、顶级律所及头部企业法务中心。
- **特质**：法律思维清醒、性格坚韧、能在复杂关系中保持定力。

# 就业与升学

- **主要流向**：西南地区大中型国企、事业单位及知名民企。

# 张雪峰风格点评

西政在法律圈就是个“传说”。它虽然不是 985/211，但你在律所或法院里，西政的师兄师姐能把你照顾得妥妥的。这里的牌子就是“实战”的保障。千万别拿它当普通市属看，这是披着市属皮的“行业巅峰”。买的是“江湖地位”。
