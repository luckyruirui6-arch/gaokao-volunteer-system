# 学校定位

重庆市属公办本科院校。被誉为“中国信息通信人才的摇篮”。在 3G/4G/5G 通信标准研发及工业网络方向具有显著实力。适合想进互联网大厂、追求技术高起薪的理性考生。

# 优势专业

- **[[通信工程]]/计算机**：行业地位极高，毕业生在华为、中兴、腾讯及各大运营商中认可度极高。
- **[[自动化]]（工业互联网）**：侧重制造业数字化转型，实战机会非常多。

# 适合什么学生

- **位次段**：重庆物理类位次约在 12,000-18,000 名左右。
- **职业目标**：大型 IT 企业架构师、[[网络安全]]专家、算法工程师。

# 就业与升学

- **主要流向**：西南地区大中型国企、事业单位及知名民企。

- **专业绑定感强**：适合职业指向明确的考生，改行成本相对存在。

# 张雪峰风格点评

重邮是西南码农的“大本营”。华为每年在成渝地区招人，重邮基本是必去的第一站。如果你打算靠代码改变命运，分儿又由于种种原因上不了电子科大，来重邮，你拿到的敲门砖质量绝对是 211 级别的。买的是“行业入场卷”。
