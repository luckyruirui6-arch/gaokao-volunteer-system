# 学校定位

位于涪陵。前身是涪陵师范。适合想在渝东南片区教书的孩子。

# 优势专业

- **特色方向**：结合地方经济需求，具有较高的就业适配度。

# 适合什么学生

- **特质**：务实、追求稳健就业、倾向留渝发展。

# 就业与升学

- **主要流向**：重庆及西南地区相关行业单位、教研及基层管理岗位。

# 报考风险

- **地域限制**：在成渝地区认可度高，跨出西南地区的品牌溢价相对减弱。

# 张雪峰风格点评

榨菜之乡的师范明珠。在渝东南那一块，这儿出来的老师最靠谱。买的是“铁饭碗路线”。
