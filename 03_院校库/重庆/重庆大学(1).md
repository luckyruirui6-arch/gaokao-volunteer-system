# 学校定位

教育部直属公办本科院校，国家“双一流”建设高校。作为西南地区的工科重镇，“建筑老八校”之一。重大在机械、电气、建筑及土木领域具有全国领先优势。适合志向高远、具备深厚数理基础且希望在西南及成渝双城经济圈发展的顶尖考生。

# 优势专业

- **电气工程（高电压）**：全国顶尖水平，深度参与国家特高压输电研发。
- **[[建筑学]]/土木/城规**：历史积淀极其深厚，是建筑圈公认的一梯队院校。
- **机械/[[车辆工程]]**：结合重庆汽车工业基地，在底盘控制与智能制造方向极具实力。

# 适合什么学生

- **位次段**：重庆物理类位次通常在前 2,000-5,000 名以内。
- **职业路径**：各级电力设计院、知名建筑事务所、大型汽车集团研发中心及政府职能部门。
- **特质**：务实、硬朗、具备极强的解决复杂系统问题的能力。

# 就业与升学

- **行业地位**：在西南电力及建筑圈，重大的文凭就是“硬通货”。
- **成渝联动**：作为成渝双城经济圈的教育旗舰，毕业生在大西部地区的职位选择面极广。

# 报考风险

- **校园地理分散**：拥有沙坪坝及虎溪等多个校区，需平衡新旧校区的学习与生活体验。
- **专业内卷**：部分王牌工科由于历史太悠久，保研及绩点竞争压力处于高压状态。

# 张雪峰风格点评

重大是重庆的“定海神针”。如果你在西南这一片想混出个名堂，重大就是最好的台阶。它的电气和建筑那是金字招牌。别听那些闲话，重大的底蕴和目前的投入产出比在 985 里是非常靠谱的。买的是一份“西南王”的底气。
