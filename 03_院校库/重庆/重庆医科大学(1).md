# 学校定位

西南地区最重要的医学人才培养基地之一。具有卓越的附属医院集群支撑。

# 优势专业

- **[[临床医学]]/儿科学/传染病学**：学科底蕴极其深厚，在西南医疗系统认可度极高。

# 适合什么学生

- **特质**：务实肯干，追求特定行业职业门槛。

# 就业与升学

- **主要流向**：西南地区大中型国企、事业单位及知名民企。

在重庆当医生，重医是“正统”。如果你分儿差一点够不上华西，来重医绝对是本地医疗圈的王道选择。买的是本地就业的“免检证”。
