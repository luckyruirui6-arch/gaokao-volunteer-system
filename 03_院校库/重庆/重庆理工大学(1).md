# 学校定位

原兵器部直属院校，“兵工七子”成员之一。在精密位移传感器、新能源汽车传动领域具有顶尖技术。适合关注智能制造与军工背景的考生。

# 优势专业

- **[[车辆工程]]/测试技术**：拥有国家级工程研究中心支撑，在精密计测方向全国领先。

# 适合什么学生

- **特质**：务实肯干，追求特定行业职业门槛。

# 就业与升学

- **主要流向**：西南地区大中型国企、事业单位及知名民企。

重工大是重庆汽车工业的“大脑”。如果你以后想去长安、赛力斯搞研发，重理工的背景非常硬。买的是“精密和严谨”。
