# 学校定位

重庆市属公办本科院校。在山区路桥、航道港口及大跨度桥梁建设方向处于全国领先地位。适合务实且能在基建一线攻坚克难的考生。

# 优势专业

- **桥梁与隧道工程**：依托重庆“桥都”地利，实战研究处于国内巅峰。
- **港口航道与海岸工程**：针对山区河流与高峡平湖的综合治理极具特色。

# 适合什么学生

- **特质**：务实肯干，追求特定行业职业门槛。

# 就业与升学

- **主要流向**：西南地区大中型国企、事业单位及知名民企。

# 报考风险

- **专业绑定感强**：适合职业指向明确的考生，改行成本相对存在。

# 张雪峰风格点评

重交是“大基建”的尖兵。如果你以后想在中国路桥、中国交建工作，重交的牌子非常管用。它是那种越老越香的专业校。买的是一份“手艺”。
