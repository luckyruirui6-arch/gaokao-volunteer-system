# 学校定位

地处重庆。中国八大美院之一。以“实验艺术”和“写实风格”著称。适合有卓越灵感、追求艺术顶层的考生。

# 优势专业

- **特色方向**：结合地方经济需求，具有较高的就业适配度。

# 适合什么学生

- **特质**：务实、追求稳健就业、倾向留渝发展。

# 就业与升学

- **主要流向**：重庆及西南地区相关行业单位、教研及基层管理岗位。

# 报考风险

- **地域限制**：在成渝地区认可度高，跨出西南地区的品牌溢价相对减弱。

# 张雪峰风格点评

川美是西南艺术的“珠峰”。那个坦克仓库和校园环境就是艺术本身。买的是这张“艺术家的名片”。
