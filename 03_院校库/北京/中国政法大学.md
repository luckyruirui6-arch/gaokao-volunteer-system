# 学校定位

教育部直属公办本科院校。法大被誉为“中国[[法学]]教育的最高学府”，是中国[[法学]]教育的龙头大校。作为“五院四系”的带头人，其在法律学术研究与司法实务领域具有深不可测的影响力。适合严谨冷静、具备高度公平正义感且立志法律事业的考生。

# 优势专业

- **[[法学]]**：绝对垄断性优势，教材编撰者及司法解释参与者多出自该校。
- **政治学与行政学**：侧重国家治理与公共管理应用，是考公进阶的优质平台。
- **[[社会学]]/法[[心理学]]**：结合[[法学]]背景的特色学科，具有较强的交叉应用价值。

# 适合什么学生

- **位次段**：北京位次通常在 2,000-4,000 名左右（[[法学]]专业组位次极高）。
- **职业路径**：法检系统公务员、顶级内资/红圈律所律师、大型企业法务及高端纠纷争议解决专家。
- **特质**：记忆力强、逻辑推导缜密、性格坚韧、能在复杂人性面前保持原则。

# 就业与升学

- **考公之王**：[[法学]]毕业生的考公通过率全国领先，进入最高检、最高法及司法部的人数占比极高。
- **行业准入**：在律师界，法大的校友证就是一份强有力的信用背书。

# 报考风险

- **专业度极高**：[[法学]]是非常辛苦的学科，必须做好“司考之苦”和长期业务磨练的打算。
- **改行成本高**：由于专业度太深，如果不从事法律行业，在其他纯商业领域的部分优势难以完全发挥。

# 张雪峰风格点评

如果你家孩子想当法官、检察官、大律师，法大就是那个唯一的、最硬的选择。在法律圈，法大人说话是带份量的。别纠结它是 211 还是 985，在[[法学]]这个圈子里，它就是 985 里的 985。但也要提醒：[[法学]]是典型的“先苦后甜”，前几年实习和律所助理的日子会非常清贫且辛苦，选这行就得有长跑的心理素质。
