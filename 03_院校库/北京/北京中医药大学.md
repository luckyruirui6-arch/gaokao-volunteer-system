# 学校定位

教育部直属公办本科院校。北中医是中国中医药高等教育的一面旗帜，在推行中医药现代化、国际化及高端研究领域处于核心地位。适合对方剂针灸、传统医学文化有深厚情怀且立志医者仁心的考生。

# 优势专业

- **中医学（九年制/五年制）**：中医领域的最高培养规格，毕业生在京内中医院认可度及学术背景极佳。
- **中[[药学]]**：侧重中药资源开发、质量控制及现代制药技术。
- **中西医结合**：培养具备现代医学思维与传统治疗手段的复合型临床人才。

# 适合什么学生

- **位次段**：北京位次波动大，高分段考生多冲九年制，低分约在 15,000-25,000 左右。
- **职业路径**：中医院系统医生、大型中药企业研发、考公进入卫生管理部门。
- **特质**：中文素养好、具备哲思能力、能背诵大量古籍经典且在医学之路上有长跑耐力。

# 就业与升学

- **医院入职率**：在中医药行业及北京本地中医院招聘中占比极高，是名副其实的行业核心。
- **多元化**：结合大健康产业，毕业生在康复养老、药膳调理等新兴领域有较多机会。

# 报考风险

- **就业滞后风险**：中医成长极慢，本科毕业往往难以独立行医，必须读硕读博且积累临床经验。
- **社会争议感知**：需考生及家长本人对中医价值高度认可，否则面对社会舆论压力可能产生职业怀疑。

# 张雪峰风格点评

北中医是中医界的“北大”。你想学最正宗的中医、以后在北京当个名老中医的传人，这里是唯一之选。它现在的九年制能让你毕业直博，这就是顶级福利。但我得提醒：学中医要背的东西不比[[临床医学]]少，甚至更玄奥，如果你是一个极度纯粹的理工科思维，可能在这儿读得会怀疑人生。这学校，是给有缘人、有悟性的人读的。
