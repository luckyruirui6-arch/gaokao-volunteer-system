# 学校定位

中央直属公办本科院校，国家“双一流”建设高校。作为中国最具代表性的综合性研究型大学之一，北大在基础学科、人文社科及医学领域处于全国领先地位。适合位次极高、具备深厚学术潜质与人文精神的考生。

# 优势专业

- **人文社科全线**：[[哲学]]、[[社会学]]、中文、历史等学科底蕴极深。
- **理学基础**：数学、物理、化学等学科在全国学科评估中长期位居前列。
- **经济与管理**：光华管理学院及国家发展研究院在国内商科领域具有极高的社会认知度。
- **[[临床医学]]**：依托北大医学部及其周边附属医院，医疗资源极其丰富。

# 适合什么学生

- **位次段**：北京位次通常在全省前 100-200 名（部分专业组可能略有浮动）。
- **职业目标**：倾向于高端学术研究、政界精英、顶级智库领军人才及大型知名企业职能管理岗。
- **特质**：思维活跃、极度自律、具备极强的自主学习能力和广阔的眼界。

# 就业与升学

- **深造导向**：毕业生出国深造及国内保研比例极高，深造质量位居全球前列。
- **行业分布**：广泛分布在国家部委、高校科研院所、顶级金融机构及互联网巨头核心部门。

# 报考风险

- **专业内部博弈**：顶尖院校内部专业分化依然存在，部分基础性较强的冷门专业就业面相对较窄。
- **高压竞争环境**：汇聚了全国顶尖生源，同辈竞争压力大，需具备良好的心理调节能力。
- **城市生活成本**：北京核心区高昂的生活成本也是长期职业发展需要考量的因素。

# 张雪峰风格点评

北大不仅仅是一个学校，它是一块金字招牌。只要你能进去，你接触到的资源、人脉和平台就是国内最顶级的。但提醒一句：别以为进了北大就能躺平。这里的专业分化很严重，如果你选了一个纯学术方向但又没有读博打算，毕业时的落差感可能会让你怀疑人生。买的是牌子，拼的是后劲。
