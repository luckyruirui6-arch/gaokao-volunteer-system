# 学校定位

教育部直属公办本科院校。前身为著名的北京地质学院。地大（北京）在地球科学、资源勘查、地质工程及珠宝首饰鉴定等领域具有显著优势。适合对地球秘密有好奇心、追求地学专业深度及珠宝奢侈品行业的考生。

# 优势专业

- **地质学/地质资源与地质工程**：全国唯二的地学殿堂，多次获得国家级科技大奖。
- **宝石及材料工艺学**：圈内著名的“地大珠宝”，在该领域具有证书颁发及鉴定话语权。
- **地下水科学与工程/海洋科学**：结合资源安全与环境治理，在前沿探索方向有深厚底座。

# 适合什么学生

- **位次段**：北京位次约在 22,000-32,000 名左右。
- **职业路径**：自然资源系统事业单位、大型矿产集团、珠宝品牌运营及鉴定中心、大型道路桥梁前期勘测岗。
- **特质**：身体素质好、具备长期野外调研的耐心、观察力极其细致。

# 就业与升学

- **行业认可度**：在地勘、资源调查及宝玉石行业，地大头衔是行业准入的“金字标准”。
- **综合保送优势**：作为 211 名校，保研率和推免质量保障了其优秀学生的学术上行空间。

# 报考风险

- **特殊作业环境**：地学专业往往涉及风餐露宿的野外实习和工作，需平衡对生活品质的主观预期。
- **职业圈层小众**：若不当研究员或不进地勘系统，部分理论地学专业的直接变现路径相对较长。

# 张雪峰风格点评

地大（北京）被很多人视为“冷门 211”，我想说那是你没看懂它带给你的“行业壁垒”。特别是它的珠宝专业，那是多少女孩子的梦想；其地球科学更是国家战略的核心。如果你孩子想在北京海淀学院路这个金子地段深扎，分数又只有二万名，地大就是那个能让你拿 211 换未来的地方。别听那些不懂行的黑地质，现在的智慧矿山和资源卫星可全是高端活儿。
