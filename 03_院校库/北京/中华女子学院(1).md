# 学校定位

中华全国妇女联合会（全国妇联）直属公办本科院校。旨在培养具有国际视野、法治意识、具有高度社会责任感的高素质女性应用型人才。适合关注性别平等、社会治理、文化传媒及希望在相对和谐的单性别环境中成长的女生（部分专业招收男生）。

# 优势专业

- **社会工作**：全国首批设立点，深度参与妇幼权益保护与社区治理研究。
- **[[法学]]（妇女权益保护）**：在婚姻家庭法及反家暴法领域有较强的行业底蕴。
- **[[学前教育]]**：侧重幼儿心理及家庭教育，在京内幼教市场有稳健的就业基础。

# 适合什么学生

- **位次段**：北京位次约在 25,000-35,000 名左右。
- **职业目标**：各级妇联组织、民政系统、社会福利机构、媒体编辑及教研岗位。
- **特质**：性格耐心细致、有同理心、关注社会民生、具备基本的文字及调研素质。

# 就业与升学

- **行业垂直度**：在特定的社工及妇幼事务岗位具有“首选性”优势。
- **京内就业**：由于地处亚运村附近，在朝阳及周边区域的社会化就业也较为便利。

# 报考风险

- **规模有限**：相比综合校，校园资源和研究平台相对较小。
- **性别环境局限**：虽对男性开放，但比例极低。对于追求社交多元化的学生需提前做心理建设。

# 张雪峰风格点评

女院在网上被很多家长低估了，其实它是“麻雀虽小五脏俱全”。如果你家孩子分儿刚过本科线不太多，又想读个带“国字头”背景的学校，女院是个非常踏实的选择。特别是它的社会工作，现在是国家重点发展的方向。这学校管理相对严格，环境单纯，适合想安静学一点务实本领的女孩子。
