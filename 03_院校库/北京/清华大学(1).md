# 学校定位

中央直属公办本科院校，国家“双一流”建设高校。清华是中国工科实力的标杆，同时在理学、经管、艺术及医学方向也具备顶尖影响力。适合致力于科学技术创新、工程领军及高端管理人才培养的考生。

# 优势专业

- **顶尖工科集群**：计算机、[[自动化]]、土木、机械、能源等学科不仅国内领先，在国际上也有极高排位。
- **经管学院**：国内商科教育的重要阵地，与国际顶级商学院有深度交流。
- **理学/生命科学**：近年来在量子信息、结构生物学等前沿领域产出了大量突破性成果。
- **美术学院**：前身为中央工艺美术学院，设计学实力雄厚。

# 适合什么学生

- **位次段**：北京位次通常在全省前 100 名以内（部分选拔项目位次更高）。
- **职业目标**：致力于大国工匠、科技领军人才、顶级金融实务及公共管理事务。
- **特质**：务实、硬核、具备极强的逻辑思维能力和解决复杂工程问题的意愿。

# 就业与升学

- **就业去向**：毕业生多流向大院大所、国防军工、战略性新兴产业及顶级金融平台。
- **学术志向**：保研及出国比例处于全国极高水平，是全球顶尖高校的重要研究生生源校。

# 报考风险

- **专业饱和感**：部分传统工科在当前的产业升级背景下，可能需要跨学科融合（如土木+[[人工智能]]）。
- **同辈压力(PEER PRESSURE)**：身边天才如云，对于心理承受能力较弱的学生，可能会感到挫败。
- **选拔限制**：部分顶尖专业（如姚班、智班）需二次选拔，竞争更加激烈。

# 张雪峰风格点评

清华是“理工男”的最终梦想，但它现在的综合实力是一点死角都没有。如果你能进清华，你的下限就是别人的上限。但我建议家长们：哪怕在清华，也要关注专业的“变现能力”。如果你孩子进了非常基础的农学或者冷门环境，在面对那些计算机、金融的同学时，这种阶层跨越的阻力依然存在。清华给了你门票，怎么演得看你自己。
