# 学校定位

教育部直属公办本科院校。中传被誉为“信息传播教育的最高学府”，是中国广播电视、新闻出版及数字媒体人才的梦工厂。适合具备卓越创意才华、敏锐信息嗅觉且立志于内容产业、播音主持及传媒前沿科技的考生。

# 优势专业

- **[[新闻学]]/传播学**：国内学科评估第一梯队，掌握主流媒体的话语权底牌。
- **播音与主持艺术**：全国标杆，培养了大量的央视及省部级电视媒体名嘴。
- **数字媒体技术/[[动画]]**：科技与艺术的深度结合，在游戏策划、视效制作方向实力强劲。

# 适合什么学生

- **位次段**：普通批位次约在 5,000-12,000 名（部分专业组差异大）。
- **职业路径**：各大电视台/电台、主流新闻门户、短视频平台、大型游戏商及品牌公关部门。
- **特质**：外向、善表达、具备审美能力、对新技术和流行文化极具感受力。

# 就业与升学

- **行业入驻**：在各大主流媒体系统中，中传人占据了大量内容生产的关键岗位。
- **新媒体红利**：在当前的视频化、社交化传播浪潮中，中传毕业生的创意产出能力极具市场溢价。

# 报考风险

- **行业红利收缩**：传统广电转型压力大，毕业生需具备极强的多媒体适应能力。
- **专业内卷**：艺术类校考入读极难，普通批入读的非核心专业需防范就业平台错位。

# 张雪峰风格点评

中传，那就是传媒人的“黄埔军校”。如果你想成名为“大V”、想进大厂搞内容，来这儿就对了。这里给你的不仅是文凭，是那种在全中国传媒圈“刷脸”的底气。但我建议家长们：传媒这行门槛在降，光能说会道没用了，一定要让孩子在学校里把底层技术（如剪辑、算法、策划逻辑）学透，否则容易在这个吃青春饭的行业里掉队。
