# 学校定位

教育部直属公办本科院校。前身为著名的北京矿业学院。矿大（北京）侧重于深地探测、矿业安全、测绘地理及资源高效利用方向。在地矿油行业具有极高的声望和人脉资源。适合务实、追求高就业保障且关注国家能源资源安全的考生。

# 优势专业

- **矿业工程**：国家级重点学科，在深部开采及防灾减灾领域处于世界前沿。
- **安全科学与工程**：跨行业高适用性学科，不仅在矿山，在化工、消防及[[工业设计]]领域也有核心优势。
- **[[测绘工程]]/地质资源与地质工程**：支撑国家大型基建与资源普查，数字化程度日趋提升。

# 适合什么学生

- **位次段**：北京位次约在 18,000-26,000 名左右。
- **职业路径**：国家能源集团、中煤集团等能源巨头核心部门、安全监管部门、大型工程设设计院。
- **特质**：求职导向务实、不畏惧去大基地、数理及工程能力优秀。

# 就业与升学

- **保底能力极强**：矿业类专业毕业生供需比长期处于高位，在能源领域几乎能保证 100% 高质量就业。
- **校友圈层**：在国家大型能源国企，矿大的校友网是毕业生职场起飞的隐形翅膀。

# 报考风险

- **专业传统感**：采矿、地质等专业名称往往给人以环境艰苦的刻板印象，需平衡个人生活追求。
- **跨界竞争**：如果不进入能源系统，该校的 211 名头在通用金融互联网领域的溢价相对有限。
