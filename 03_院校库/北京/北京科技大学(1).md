# 学校定位

教育部直属公办本科院校。前身为著名的北京钢铁学院，被誉为“钢铁摇篮”。北京科技大学在材料科学、冶金工程及矿业工程领域处于全国学术之巅，近年来在机械、[[人工智能]]、科技史等方向也取得了跨越式发展。适合稳健务实、立志于大国工程材料与先进制造的考生。

# 优势专业

- **[[材料科学与工程]]**：全国顶尖水平，深度参与国家航天、航母及先进光刻材料研发。
- **冶金工程**：国家重点学科，在绿色冶金及新金属材料方向引领行业。
- **[[自动化]]/智能制造**：结合工业背景，在工程控制与机器人研发领域优势明显。

# 适合什么学生

- **位次段**：北京位次约在 5,000-9,000 名之间。
- **职业目标**：大型国企研发中心、顶级制造企业（华为/商飞）、高等院校及科研院所。
- **特质**：能耐得住寂寞做实验、逻辑推导能力强、具备较强的工程创新意识。

# 就业与升学

- **深造导向**：具有保研及出国优势，由于是老牌部属高校，背景在科研圈认可度极高。
- **就业韧性**：在实体经济回归、材料强国的背景下，毕业生被“抢人”现象时有发生。

# 报考风险

- **行业艰苦感**：部分传统矿冶专业可能需要进入生产现场，对于希望纯白领办公的学生需提前规避。
- **专业分配风险**：热门专业如计算机、材料实验班分数溢价较高。

# 张雪峰风格点评

北科大，名字听着没以前响亮，但它是“内功极深”的高校。它的材料和冶金在全国那是横着走的。如果你孩子想学务实的技术，想以后进国家部委下面的重点国企平台，北科大就是最好的跳板。它现在也搞了很多跨学科的精英班，分数不低，填志愿时要关注位次，别被简单的校名给低估了。
