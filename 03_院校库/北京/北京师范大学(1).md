# 学校定位

中央直属公办本科院校，国家“双一流”建设高校。北师大是中国师范类院校的最高学府，在教育学、[[心理学]]、中国语言文学及地理学领域处于全国垄断性领先地位。适合立志于成为顶尖教育家、[[心理学]]者及高端人文社科研究者的考生。

# 优势专业

- **教育学/[[心理学]]**：全国学科评估连续排名第一，代表了国内最高研究水平。
- **中国语言文学/中国史**：文史底蕴深厚，培养了一大批人文学术大师。
- **地理学/生态学**：理学方向在京内及全国均有极强竞争力。
- **数学/物理**：基础学科实力稳健，是北京教育系统高端师资的输出地。

# 适合什么学生

- **位次段**：北京位次通常在 1,500-3,500 名左右。
- **职业目标**：顶尖公办学校教师（含高中/大学）、教育政策制定者、高端心理咨询机构、学术研究机构。
- **特质**：性格稳重、热爱教学与研究、具备较强的人文情怀与沟通能力。

# 就业与升学

- **教育系统影响力**：北师大的校友几乎覆盖了全国各地知名中学的管理层。在北京教育系统，北师大的招牌就是公认的“第一优先级”。
- **多维发展**：由于其综合实力强，除当老师外，在传媒、行政管理、互联网人力方向也有较强表现。

# 报考风险

- **专业吸引力不均**：部分理学或农学方向在大环境下可能受到忽视，需结合个人兴趣填报。
- **师资过剩隐忧**：随着高端师范生源爆炸，部分特大城市（如北京/上海）的三甲附属学校入职门槛已攀升至博士学历。

# 张雪峰风格点评

如果你孩子想当老师且分够，全中国只有一所学校叫北师大。它不是普通的师范，它是“双一流”里的全能选手。它教的不止是知识，还有“怎么育人”。但要说句实在话：别一进北师大就想着一劳永逸，现在的顶级中学招聘，你要没个研究生学历、没几个漂亮的实习，光有本科学历也是很难进名校编制的。北师大是你的高起点，不是你的终点站。
