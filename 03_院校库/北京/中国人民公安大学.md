# 学校定位

公安部直属公办本科院校。被誉为“共和国警官的摇篮”，是中国公安教育的最高学府。在犯罪学、刑事技术、治安学及[[网络安全]]执法领域居全国垄断地位。适合身体素质卓越、具备正义感且立志参警的考生。

# 优势专业

- **侦查学/治安学**：全国学科建设标杆，师资力量及行业资源处于公安系统巅峰。
- **刑事科学技术**：结合现代法医、物证提取及生物信息技术，实战性极强。
- **[[网络安全]]与执法**：针对新型网络犯罪开展的防攻研究，在行业内极具话语权。

# 适合什么学生

- **录取模式**：本科提前批（零批次），需参加各省公安厅组织的体检、政审、体能测评及面试。
- **位次段**：北京位次约在 5,000-10,000 名左右（男生分数通常略低于女生）。
- **特质**：绝对忠诚、纪律性强、能适应警务化管理作息、身体协调性好。

# 就业与升学

- **公安联考红利**：参加针对公安院校毕业生的内部招警考试，入警率常年保持在 90% 以上。
- **职业路径**：广泛分布在公安部、各级公安厅（局）及实战单位。

# 报考风险

- **体检与政审红线**：视力（含矫正）、身高、纹身、家族成员背景不合格者直接淘汰。
- **专业定向感**：如果由于健康或纪律原因离警，专业知识在社会普通企业的复用性相对有限。

# 张雪峰风格点评

公大是警察里的“北大”。如果你家孩子将来想穿警服、想进公安部或者是去北京市局当警察，选公大绝对没错。现在的入警率摆在那儿，就是给孩子一个确定的未来。但我提醒一下：警校没有传说中那么“飒”，每天几点起床、被子叠成什么样都有规矩，想自由散漫的孩子千万别来。
