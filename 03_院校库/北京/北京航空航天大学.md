# 学校定位

工业和信息化部直属公办本科院校，国家“双一流”建设高校。北航是中国航空航天科技最高水平的代表之一，在计算机、先进制造、材料及[[自动化]]等领域处于工业前沿地位。适合具备强烈科技报国志向、数理基础扎实的理科精英。

# 优势专业

- **航空宇航科学与技术**：全国顶尖水平，深度参与国家重大航天项目。
- **[[计算机科学与技术]]/[[软件工程]]**：在[[人工智能]]、[[网络安全]]方向具有极其硬核的科研实力，就业薪酬稳居全国前列。
- **[[自动化]]/仪器科学与技术**：在精密制造与控制理论方向办学积淀深厚。

# 适合什么学生

- **位次段**：北京位次约在 1,000-2,000 名之间。
- **职业规划**：致力于航空航天、国防科技、大型互联网大厂研发或高端战略性新兴产业。
- **特质**：思维严谨、抗压能力强，能适应高强度的工程技术研发工作。

# 就业与升学

- **行业对口**：毕业生大量流向中国航天科技集团、航空工业集团等国防工业核心单位，就业稳定性与行业门槛极高。
- **互联网大厂**：北航计算机系毕业生是百度、网易、华为等企业的核心招聘对象。

# 报考风险

- **军工保密限制**：部分涉密专业或项目可能对出国交流及后续私人企业求职有一定影响（如限制出境清单）。
- **学科偏向性**：虽然也在发展文理医，但北航的“灵魂”依然是工科，文科生在这里的社团及学术体验可能不占主导。

# 张雪峰风格点评

北航是目前的“工科红人”。如果你家孩子想搞硬核科技、想以后去航天院或者是去华为大厂拿高薪，北航的性价比甚至是超过某些老牌985综合校的。它就是两个字：硬核。但要提醒一下：北航的学习压力不是开玩笑的，基本全是硬骨头，如果你没那个钻研劲儿，可能这四年会读得很痛苦。
