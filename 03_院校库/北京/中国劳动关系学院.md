# 学校定位

中华全国总工会直属公办本科院校。旨在培养具有宏观劳动关系视野、能处理复杂社会劳动纠纷及精通工会事务的高素质应用型人才。适合关注劳方权益、追求职业稳定且关注[[人力资源管理]]的考生。

# 优势专业

- **劳动关系/劳动与社会保障**：全国首创且唯一具有完整学科链的方向，行业认可度垄断。
- **[[人力资源管理]]**：结合工会背景，在劳资协商、绩效评价方向极具特色。
- **[[法学]]（劳动法方向）**：针对劳动争议解决的细分[[法学]]，在企业合规领域吃香。

# 适合什么学生

- **位次段**：北京位次约在 20,000-30,000 名左右。
- **职业目标**：各级工会组织、大型国企的人资/党群部、劳动仲裁机构或法务专员。
- **特质**：具备同情心、公正心、沟通技巧好、能适应较琐碎的协调事务。

# 就业与升学

- **工会系统保障**：每年有大量毕业生进入全国各地的总工会及产业工会系统。
- **国企偏好**：大中型国企在招聘 HR 时，非常看重该校毕业生的实践与专业匹配度。

# 报考风险

- **专业垂直度极高**：如果不做 HR 或不去工会，专业知识的行业壁垒转换需要一定的努力。
- **校园氛围平实**：相对朴素，追求“先锋艺术”或“极限科技”氛围的学生可能不适应。

# 张雪峰风格点评

劳动关系学院是一所“稳得不能再稳”的学校。如果你分儿一般，以后想进国企干个 HR、或者是回老家干个公务员，这儿的专业适配度高得吓人。它虽然不是双一流，但在处理“人与人、人与活儿”的关系上，它是全国第一。这是一所适合务实家庭、想让孩子早点在社会系统里落座的良心校。
