# 学校定位

由中国社会科学院直接创办的一所“精而专”的大学。依托社科院庞大的研究资源，实行小班化、精英化培养，旨在培养引领未来人文社会科学发展的顶尖研究人才。适合热爱纯学术研究、具备高度理论洞察力的文科精英。

# 优势专业

- **人文社科全线覆盖**：[[哲学]]、经济学、[[法学]]、[[新闻学]]及历史学均由顶级研究员授课。
- **导师制**：本科生即配备学术导师，接触前沿科研项目的机会处于国内极高水平。
- **学术平台**：背靠国家最高级的人文社科研究机构，图书及数据资源极其优越。

# 适合什么学生

- **位次段**：北京位次约在 1,000-2,500 名左右（由于招人少，分极稳）。
- **职业目标**：顶尖高校教席、国家级智库研究员、公共政策制定者及大型传媒高管。
- **特质**：能耐得住学术寂寞、文字功底扎实、思维具有高度深度与批判性。

# 就业与升学

- **深造导向**：保研率极高，且毕业生在考取本院及其他 985 院校研究生时具有极强的声望优势。
- **行业影响**：虽然在传统商业市场名气相对内敛，但在体制内及学术界认可度极高。

# 报考风险

- **校园生活小众**：学生人数少，社团及校园活动丰富度不如大型综合校。
- **地理位置**：良乡校区距离中心城区较远，需适应安静的学术氛围。
- **专业窄口径**：理科专业目前极少，几乎不适合纯理科方向的考生。

# 张雪峰风格点评

社科大是文科生的“保研神校”。如果你家孩子一门心思想要做研究、写论文、以后进高校当教授，这儿的资源甚至比很多老牌 985 都硬。因为它直接就是给社科院选拔苗子的。但也得说，这里不适合那种想过“红火”大学生活的人，这儿是做学问的苦行僧待的地方。
