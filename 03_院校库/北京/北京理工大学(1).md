# 学校定位

工业和信息化部直属公办本科院校，国家“双一流”建设高校。学校前身为自然科学院，是著名的“国防七子”成员。在武器系统、智能制造、网络空间安全及光学工程领域具有深厚的红色基因和硬核实力。适合务实、低调且追求顶尖工程技术的考生。

# 优势专业

- **兵器科学与技术**：国家级顶尖学科，代表了我国在相关领域的尖端科研水平。
- **控制科学与工程/机械工程**：在坦克装甲车辆、特种机器人等方向优势显著。
- **计算机/[[通信工程]]**：在信息安全、特种通信方向办学特色鲜明，就业前景广阔。

# 适合什么学生

- **位次段**：北京位次约在 1,500-3,000 名左右。
- **职业规划**：倾向于国防工业、重点科研院所及高端智能制造企业。
- **特质**：踏实肯干、数理基础优秀，具备攻坚克难的意志力。

# 就业与升学

- **行业背景**：作为兵器工业的人才摇篮，与中国兵器工业集团、兵器装备集团等有极其紧密的合作关系。
- **社会化就业**：在智能驾驶、[[网络安全]]等社会化热门领域也非常受名企青睐。

# 报考风险

- **行业敏感性**：部分敏感专业可能涉及“实体清单”风险，对后续申请海外留学（特别是北美）有一定影响。
- **男女比例**：传统理工校底色，校园氛围相对理性简洁，人文活动相对少一些。

# 张雪峰风格点评

北理工就是典型的“实干派”。它虽然名字带“理工”，但它在国防工业里的份量是常人难以想象的。如果你想让孩子学一门带“护城河”的技术，以后进重点军工单位端铁饭碗，北理工就是首选。它现在的徐特立班等选拔项目分很高，竞争非常激烈。这是一所能让孩子学到“真家伙”的学校。
