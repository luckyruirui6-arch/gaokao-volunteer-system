# 学校定位

北京市属公办艺术院校，中国戏曲教育的最高学府。学校以京剧表演为龙头，涵盖地方戏、舞台美术、戏曲导演等全产业链。适合从小习艺或对传统文化艺术具有深厚兴趣的考生。

# 优势专业

- **戏曲表演（京剧）**：国家级特色专业，聚集了戏曲界的顶尖名师。
- **戏剧影视美术设计**：在传统与现代舞台美术结合方向具有特色。
- **戏曲导演/作曲**：针对戏曲本体的创作与教学体系。

# 适合什么学生

- **招生特点**：高度专业化。多数表演方向要求具备扎实的基本功（童功）。
- **职业目标**：各大京剧院团、地方剧团或戏剧研究机构。
- **情怀导向**：适合立志于传承中华传统文化的学生。

# 就业与升学

- **定向培养感**：许多毕业生直接对接国家及地方重点剧院，属于窄口径、高壁垒就业。
- **行业分布**：由于专业度极高，转行难度相对较大，但行业核心岗位不可替代。

# 报考风险

- **市场环境**：传统戏曲受众相对分众化，商业化程度不如影视表演。
- **身体损耗**：表演类专业对体力及身体素质要求极高，存在一定的职业损耗风险。

# 张雪峰风格点评

国戏是一块“金字招牌”，但它非常挑人。如果你不是真的爱戏、懂戏，或者没有那个童子功，在这儿是待不住的。但如果你想在传统艺术这条路上走到底，这里就是你能找到的最好的平台。这是一份需要耐得住寂寞、守得住情怀的事业。
