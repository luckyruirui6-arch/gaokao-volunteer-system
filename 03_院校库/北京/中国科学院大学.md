# 学校定位

高度专业化的公办研究型大学，由中国科学院直属领导。国科大秉承“科教融合”理念，以博士生培养起家，目前开展精英化本科招录，旨在培养未来的顶尖科学家。适合具备极高数理天赋、立志终身从事基础科研的一流考生。

# 优势专业

- **全明星导师配置**：本科生实行导师制，师资阵容包含大量两院院士及顶级研究员。
- **理学基础学科**：数学、物理、生物、天文等学科的科研环境堪称国内“实验室天花板”。
- **前沿交叉方向**：在量子科技、深海探测、[[人工智能]]底层算法等领域有近水楼台的科研优势。

# 适合什么学生

- **位次段**：北京位次通常在全省前 300 名以内（竞争烈度直逼清北）。
- **职业规划**：90% 以上的目标应为硕博连连读及全球顶尖实验室研究员。
- **特质**：极度专注、能耐得住寂寞、具备极强的数学思维和探索未知的原创力。

# 就业与升学

- **升学主导**：本科生深造率居全国首位，绝大部分学生直接进入中科学院各研究所或全球顶尖名校开展学术路径。
- **行业影响**：虽然不以“找工作”为直接目的，但毕业生在芯片底层设计、高级算法专家等高壁垒岗位具有极强竞争力。

# 报考风险

- **学习负荷极其恐怖**：课程深度远超普通大学，对于想体验“多姿多彩大学生活”或混文凭的人来说是噩梦。
- **专业局限**：基本不设人文社科、商科专业，完全不适合职业目标在财经、法律等偏应用领域的学生。
- **校区位置**：本科生主要在雁栖湖或玉泉路，相对远离市中心闹市区。

# 张雪峰风格点评

国科大，我把它定义为“科学家的摇篮”。如果你家孩子是个纯粹的理科学霸、以后想拿诺贝尔奖、想搞中国芯，那去国科大甚至是比去清华更好的选择。因为在这里，你大一就有可能跟着院士做实验。但我也得提醒：如果你只是想毕业早点赚钱养家、或者社交能力特别强想搞管理，去这儿就是受罪。它只爱天才和疯子，平庸在这儿活不下去。
