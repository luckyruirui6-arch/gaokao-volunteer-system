# 学校定位

教育部直属公办美术院校。前身为国立北平艺专，是中国现代美术教育的开端。学校在绘画、雕塑、建筑及设计学领域处于国内准垄断的审美领先地位。适合具备卓越视觉才华、立志引领时代审美与艺术创新的考生。

# 优势专业

- **中国画/油画/版画**：传统优势王牌，代表了国内纯艺术创作的最高高度。
- **设计学（[[工业设计]]/视传）**：结合现代审美与实际应用，在国家大型项目（如奥运、世博设计）中承担重任。
- **[[建筑学]]/城市设计**：侧重人文建筑与艺术空间，是国内最具艺术感的建筑教育。

# 适合什么学生

- **选拔模式**：本科提前批，需参加并通过各省联考加中外校考。
- **位次段**：对文化分要求逐年提高，北京普通批（非艺考生）招生位次可达 6,000-10,000 名以内。
- **特质**：有极强的视觉表达欲、审美敏感、不随波逐流、具备一定的[[哲学]]思辨能力。

# 就业与升学

- **艺术名声溢价**：毕业生在顶级设计公司、美术馆、出版社及艺术教研系统极其吃香。
- **自主创业领军**：由于自带“央美”品牌，其开办艺术机构或工作室的成功率在圈内最高。

# 报考风险

- **纯艺就业曲线**：油画、国画等专业在前期可能面临较长的市场探索期。
- **地域依赖**：央美的影响力极大集中在一线城市的高端艺术消费市场。

# 张雪峰风格点评

央美是一所“有灵魂”的学校。它是中国美术界的旗帜。如果你分挺高，又真的爱画画，想以后作品能放进美术馆，央美是唯一的去处。现在的央美对文化课越来越看重，那些“想靠艺考混文凭”的孩子已经进不去了。在这里，你学到的不仅是画画，更是一种观察世界的方法。这学校的牌子，真香。
