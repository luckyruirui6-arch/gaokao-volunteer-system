# 学校定位

教育部直属公办艺术类高校，中国唯一一所进入“211工程”的艺术院校。它代表了中国专业音乐教育的最高水平。适合具备卓越音乐天赋、经过长期高强度专业训练且目标是国际级音乐家或顶级乐团人员的考生。

# 优势专业

- **演奏/声乐**：涵盖古典、现代多种乐器，各系部均由国内该领域最顶尖的大师主持。
- **指挥/作曲**：国家核心创作人才的孵化中心。
- **音乐学/音乐教育**：在理论研究和高端人才培养方向具有极深底座。

# 适合什么学生

- **选拔模式**：本科提前批，需通过极严苛的校考（初试、复试、三试）。
- **特质**：极度沉浸于音乐世界、具备极强的表现欲和舞台感、能耐受长达十几年的练琴枯燥。
- **财务能力**：需考虑极高的校外培训成本及昂贵的乐器投入。

# 就业与升学

- **顶层就业**：毕业生主要进入国家级艺术团（如中交、国交）、艺术院校及国际知名演艺平台。
- **行业影响**：是国内音乐圈考级、评审及学术话语权的绝对核心。

# 报考风险

- **录取率低到离谱**：报录比常年处于几十比一甚至上百比一。
- **职业路径单一**：如果不成为“职业演奏家”，在大众市场的就业转换成本不低。

# 张雪峰风格点评

央音，是音乐生的“珠穆朗玛”。如果你家孩子不是全省数一数二的天才，或者是顶级名师手下的得意门生，想考这儿是非常难的。来这儿念书，你买到的是进入中国音乐圈顶层的门票。但也得提醒：这里的竞争是非常残酷的，它只看专业实力，不管背景。普通家庭如果只是想学个乐器，建议去综合类大学的艺术系更稳一些。
