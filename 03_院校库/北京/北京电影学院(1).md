# 学校定位

北京市属公办艺术类高校，亚洲最大的电影专业院校。学校在表演、导演、摄影及[[动画]]等领域具有显著的行业影响力。适合具备极佳的艺术表现力、创意思维且立志于投身影视工业的考生。

# 优势专业

- **表演**：社会知名度最高，培养了大量的影视演艺人才。
- **导演/电影学**：侧重电影思维转换与剧本创作，学术底蕴深厚。
- **摄影/录音**：影视技术类核心，在工业制作领域具有不可替代性。
- **[[动画]]**：结合现代视觉特效，在校产研各环节有深度合作。

# 适合什么学生

- **选拔模式**：主要在“本科提前批”录取。除文化课外，录取规则因专业而异（如专业课排名或文化课折算）。
- **职业目标**：倾向于进入影视制作公司、电视台、互联网大厂或成立个人工作室。
- **特质**：外向、富有表现力、能适应高强度的剧组工作节奏或创意产出压力。

# 就业与升学

- **行业现状**：主要流向影视娱乐业、新媒体产业及文化传媒行业。
- **隐形门槛**：影视行业不仅看学历，更看个人的作品质量、项目经历及行业人脉。

# 报考风险

- **成名偶然性**：表演等专业具有较强的不可控性，并非名校毕业就能成名成家。
- **行业波动**：影视市场受政策、平台规模影响较大，职业稳定性相对较弱。
- **备考周期**：为了校考，许多学生需要长达数年的专业集训。

# 张雪峰风格点评

北电是无数人的梦想，但它背后的竞争是极其惨烈的。不要只看到光鲜亮丽的明星学长学姐，你要看到那些在幕后熬夜剪辑、扛机器的同学们。这所学校提供的是中国电影圈最顶层的人脉网，但它不保证你的下限。如果你没有极其强烈的表现欲和创作热情，在这里会感到非常迷茫。
