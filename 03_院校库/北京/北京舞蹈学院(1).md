# 学校定位

北京市属公办艺术类高校，被誉为“舞蹈家摇篮”。学校是中国唯一的专业舞蹈高等学府，在古典舞、民间舞、芭蕾舞、音乐剧等领域处于领先地位。适合通过长期专业训练、具备卓越身体条件的舞者。

# 优势专业

- **舞蹈表演**：包含中国舞、芭蕾舞、国标舞等，对身体机能考核极其精细。
- **舞蹈编导**：侧重创作与排练逻辑，适合具备创意思维的舞者。
- **舞蹈学/通识教育**：侧重舞蹈理论研究与教育普及。

# 适合什么学生

- **报考条件**：对身高、身材比例、软开度等先天条件有近乎苛刻的要求。
- **职业目标**：顶级舞团领舞/演员、高端舞蹈教培机构或文化传媒创作中心。

# 就业与升学

- **行业地位**：毕业生几乎覆盖了国内所有的顶级舞团，是舞蹈行业的金字招牌。
- **跨界能力**：优秀的舞者在影视表演、时尚模特等领域也有较强的跨界竞争力。

# 报考风险

- **职业周期短**：纯表演方向受年龄和伤病影响较大，需提早规划教学或编研转型。
- **选拔残酷度**：从附中到本科，能够走到最后的都是凤毛麟角。
