# 学校定位

北京市属公办本科院校。首医大在北京市级医疗系统中具有较强的地区竞争力，拥有宣武、安贞、同仁、天坛等知名附属医院资源。作为市属高校中的医学高地，其在京内的医疗行业认可度相对较高。适合目标锁定在京内大型医院或卫生系统就业的应用型医学考生。

# 优势专业

- **[[临床医学]]**：学校的核心学科，在心血管、神经科学、眼科等方向具有一定的学术影响力。
- **[[口腔医学]]**：热门专业，依托附属口腔医院资源，就业方向明确。
- **[[预防医学]]/公共卫生**：在公共卫生管理和流行病调研方向有较长时间的办学积累。
- **[[医学影像学]]/[[药学]]**：侧重实践应用，与临床需求结合紧密。

# 适合什么学生

- **分数段**：北京位次在 8,900 至 25,000 左右的高分段学生（临床方向位次通常更高）。
- **职业目标**：倾向于在北京市各级公立医院或医疗卫生系统落脚的学生。
- **心理准备**：能接受医学专业较长的培养周期，具备较强的抗压能力。
- **特质**：学习自律，对医学行业有理性的认识和敬畏心。

# 就业与升学

- **就业地区**：从现有资料看，有较大部分毕业生留在北京就业。
- **就业行业**：集中在医疗卫生、制药及生物技术相关领域。
- **升学情况**：具备推荐免试研究生资格，部分表现优异的学生可前往京内顶尖医学院校或本校深造。
- **行业渊源**：作为市属医学院校，其校友广泛分布在京内医疗系统。

# 报考风险

- **学历竞争**：医学行业对高学历（硕博）有普遍要求，需做好长期学习和投入的规划。
- **调剂可能性**：临床、口腔等专业组关注度极高，分数擦边可能面临专业方向的内部调剂。
- **选科限制**：核心专业组对物理、化学有明确限制，填报时需严格核对。

# 张雪峰风格点评

如果你家孩子想当医生，且希望留在北京发展，首医大是一个务实且扎实的选择。
