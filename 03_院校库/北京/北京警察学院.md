# 学校定位

北京市属公安类公办本科院校。主要为北京市公安局及各级公安机关培养警务人才。学校实行警务化管理，具有极强的政治属性和行业垄断性。适合立志参警、身体素质好且政治忠诚的北京籍考生。

# 优势专业

- **侦查学/刑事科学技术**：公安教育的核心，理论与警务实战紧密结合。
- **治安学/[[网络安全]]与执法**：适应现代特大城市管理及[[网络安全]]治理需求。
- **交通管理工程**：针对北京复杂的交通环境开展的特色教育。

# 适合什么学生

- **录取批次**：在“本科提前批”录取，包含严格的政审、体检、体能测评及面试。
- **户籍限制**：通常仅限北京户籍考生报考（北京生源）。
- **职业志向**：铁了心要当公安民警，能接受半军事化管理的作息与制度约束。

# 就业与升学

- **入警优势**：通过公安联考，毕业生的入警率极高（通常 90% 以上），几乎是进入首都警队的捷径。
- **岗位分配**：主要分配至北京市各分局、派出所及交警、网安等实战部门。

# 报考风险

- **体检与政审**：这是硬指标，视力、身高或家族成员政审有问题会直接淘汰。
- **专业局限**：如果将来不当警察，该校专业的通用性较弱，社会化改行成本较高。
- **纪律约束**：大学四年完全没有普通高校的散漫自由，需根据性格理性评估。
