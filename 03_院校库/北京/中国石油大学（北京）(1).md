# 学校定位

教育部直属公办本科院校。被誉为“石油人才的摇篮”。石大（北京）在石油天然气工程、油气储运及地球科学领域具有核心领导力。适合志在投身国家能源勘探及其下游炼化工业、对跨国作业及能源战略感兴趣的考生。

# 优势专业

- **[[石油工程]]/油气储运工程**：全国垄断级优势，对接中石油、中石化、中海油全球全产链。
- **地球[[物理学]]**：在地下结构探测方向实力强劲，结合大数据处理向智能化转型。
- **化学工程与技术（石化特色）**：在大规模炼厂[[自动化]]及新型石油化工合成方向在业内有名。

# 适合什么学生

- **位次段**：北京位次约在 18,000-28,000 名左右。
- **职业路径**：三大油核心研发与生产管理岗、能源咨询公司、跨国能源贸易及大型工程技术服务公司。
- **特质**：具备开阔的职业边界观、能接受驻外项目、工程思维敏锐。

# 就业与升学

- **行业对口度极高**：在石油系统内部，石大文凭在招聘时几乎代表了“专业免检”。
- **跨国机会多**：中国石油企业的海外项目需求庞大，外语好、专业硬的毕业生极易获得海外轮岗机会。

# 报考风险

- **单一行业绑定**：由于行业特色极其鲜明，个人职业轨迹受石油产业景气度影响较其他行业更显著。
- **校区位置**：位于昌平区，相对远离市中心主功能区。

# 张雪峰风格点评

石大（北京）是 211 里的“资源库”。如果你家里有人在石油系统或者是想让孩子以后走国家能源战略这条路，石大就是无可争议的顶流。它现在也在转型，搞了很多碳中和、[[人工智能]]+石油，这是个可以干一辈子的行业。学石大，就要学它最硬的[[石油工程]]，那是真正的行业门票。
