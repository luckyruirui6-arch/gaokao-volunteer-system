# 学校定位

北京市属公办本科院校，国家“双一流”建设高校。首师大是北京市重要的师范人才培养基地，承担了本地较大部分中小学及幼儿园骨干教师的培养任务。它是的一所教育实践资源丰富、文理基础研究具有一定优势的综合性师范大学。

# 优势专业

- **[[小学教育]]/[[学前教育]]**：北京市属高校中具有学科优势的方向，就业对标京内公办基础教育系统。
- **中国语言文学/历史学/地理学**：国家级一流本科专业，师范方向在本地具有一定的认可度。
- **[[数学与应用数学]]**：双一流建设学科，是京内中学数学师资的重要来源之一。
- **[[心理学]]/美术学**：学科建设相对均衡，就业路径涵盖教育及相关延伸行业。

# 适合什么学生

- **分数段**：北京位次在 14,000 至 21,000 左右的学生。
- **职业目标**：倾向于在毕业后进入北京中小学、幼儿园编制系统的学生。
- **特质**：性格稳重、有耐心、表达能力较好。
- **稳定偏好**：看重职业稳定性及北京市教育系统的行业人脉。

# 就业与升学

- **就业地区**：从现有资料看，有较大部分毕业生留在北京就业。
- **就业行业**：教育行业是主要流向，也涉及文化传播、公共管理等领域。
- **升学情况**：具备推荐免试研究生资格，部分毕业生前往京内名校或国外教育学院深造。
- **行业地位**：在本地基础教育行业拥有较强的校友纽带，校友广泛分布在各级教研及管理岗位。

# 报考风险

- **编制供需动态**：需关注地区教育编制政策的调整，竞争压力可能持续存在。
- **专业属性区分**：包含师范类与非师范类专业，两者的就业逻辑与培养路径存在差异。
- **校园环境**：校区相对分散，填报阶段需核实现有专业的具体就读地点。

# 张雪峰风格点评

如果你分数进不了最顶尖的那几所，但想留在北京当老师，首师大是非常重要的选择。
它是北京中小学师资的输出地。在本地教育圈子，这里的文凭和校友人脉是非常有用的。但也要泼盆冷水：现在的师范不再是“旱涝保收”，得有真本事拿证、过面试。如果你不是真心想教书，只是想混日子，这里的竞争可能也会让你感到压力。
