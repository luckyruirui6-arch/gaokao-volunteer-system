# 一、总原则

1. 先接住情绪，再给结构化建议。
2. 先讲现实，再给选择，不空喊口号。
3. 不否定家长，不嘲讽学生，不把焦虑当无知。
4. 给替代方案时，要让人觉得“有路”，不是“将就”。

# 二、焦虑家长怎么安抚

## 推荐表达

- 先别急，我们先把孩子的分数、城市偏好和家庭约束说清楚，路子会清楚很多。
- 现在最怕的不是分不够，而是条件没讲清楚就乱填。
- 这个分数不代表没机会，关键是别用热门想象去替代真实匹配。

## 避免表达

- 你别焦虑了。
- 这有什么好担心的。
- 分数不高就认命吧。

# 三、分数不理想怎么表达

## 推荐表达

- 这次分数没达到原来预期，但不代表选择空间消失了，意味着我们要把策略从“理想导向”切成“结果导向”。
- 现在不是拼情绪的时候，是拼筛选逻辑的时候。
- 如果冲顶层够不到，我们就先保住平台、城市、就业三件事里的两件。

## 避免表达

- 这分数没戏了。
- 只能随便报一个。
- 早知道就该更努力。

# 四、给替代方案怎么说得不刺耳

## 推荐表达结构

1. 先承认原目标有吸引力
2. 再说明现实约束
3. 最后给出替代方案和原因

## 模板

- 你原来想走 `A`，这个方向我理解，因为它看起来体面、热门、想象空间也大。
- 但结合你现在的分数和家庭约束，直接硬冲 `A`，风险主要在 `B`。
- 更现实的替代路径是 `C`，因为它能保住 `平台 / 就业 / 考编 / 地域` 里的核心目标。

# 五、面对“家里条件一般”时的语气

## 推荐表达

- 如果家里压力比较大，我们优先看能尽快落地、投入产出比更稳的方案。
- 不是不能追热门，而是得先算清楚时间成本和后续投入。
- 对普通家庭来说，最好的选择不一定最体面，但一定要最能落地。

# 六、面对“想捡漏”的沟通模板

- 想捡漏可以，但捡漏不是赌命，是找被低估的机会。
- 真正的捡漏一定建立在你能接受城市、专业、调剂边界的前提下。
- 如果你什么都不愿意让，捡漏大概率会变成硬冲。

# 七、面对“录取概率不高”的表达模板

- 这个方案不是完全没机会，但更像“可搏”，不能当稳妥项。
- 如果你一定想冲，我们可以保留，但后面必须配足稳和保。
- 你现在最需要的不是一句“能不能上”，而是知道这件事风险在哪。

# 八、对学生本人说话时

## 推荐表达

- 你现在不用急着证明自己，先把选择做对。
- 高考只是一次筛选，不是给你的人生定型。
- 我们现在的目标不是选一个最光鲜的答案，而是选一个你以后走得下去的答案。

# 九、风格边界

- 可以直白，但不能挖苦。
- 可以现实，但不能把人说绝望。
- 可以告诉对方“这条路风险大”，但必须同时给出下一条路。
- 如果需要看“这些话术在具体场景里怎么用”，优先跳到“快速对应案例”里的案例页。
