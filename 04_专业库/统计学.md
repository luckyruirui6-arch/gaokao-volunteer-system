# 专业定位

**统计学**（代码：071201）是理学门类的核心专业，学制4年，授予学士学位。

# 核心课程

- 概率论与数理统计
- 多元统计分析
- 回归分析
- 抽样调查
- 时间序列分析
- R语言/SAS统计分析
- 生物统计

# 适合什么学生

- 对本领域有浓厚兴趣，能主动自学
- 逻辑思维严谨，能接受抽象概念训练
- 能接受该专业学制要求的学习节奏与强度

# 不适合什么学生

- 对数学推导极度抵触
- 只想做偏业务的分析而不愿意做模型推导


# 专业优点

- 理论基础扎实，很多方向是后续深造、跨学科发展和高门槛岗位的底层能力来源。
- 对思维能力强、愿意继续学习的人来说，后劲往往不差。
- 如果学校科研平台较强，这类专业在升学和转向上会有额外优势。

# 专业缺点

- 本科阶段的职业出口不一定像应用型专业那样直接。
- 很多方向更吃深造，不适合只想本科毕业立刻兑现的人。
- 如果学校平台一般、自己又不打算读研，前期体感可能不如应用专业直接。

# 未来前景

- 基础科学类专业的价值，很多时候不是立刻兑现，而是体现在后续深造、科研、交叉学科和高门槛技术岗位。
- 对愿意长期投入的人，这类专业并不差；对强烈追求短期回报的人，要提前讲清楚。
- 真正报考时，要先分清你想走的是“直接就业”还是“研究和深造”路线。


# 适合的家庭类型

- 能接受长期学习、深造和科研导向的家庭。
- 更看重基础能力和长期积累，不急着本科毕业立刻变现的家庭。
- 愿意支持孩子后续考研、读博或转向交叉学科的家庭。

# 发展路线

- 本科阶段先把基础课和数理逻辑打牢。
- 如果想做研究，就尽早找科研平台和导师。
- 如果不走纯科研，也可以往数据、算法、材料、应用研发等方向转。

# 张雪峰风格点评

基础和交叉科学类专业，最现实的一点就是：本科阶段通常只是起步，后面读研的权重会明显上来。它们适合能坐得住、愿意钻研、对数学和实验不排斥的人。家里如果接受孩子走学术、研发或者技术型路线，这类专业是有价值的；如果想本科就直接变现，那预期要放低。

# 就业方向与薪资参考

> [!NOTE]
> 薪资数据基于行业共识与公开调研，实际因地区、学校层次、个人能力差异显著。

## 主要就业方向

- 生物医药数据统计师（CRO公司）
- 金融风险管理
- 市场调研分析师
- 政府统计局
- 数据分析师
- 保险精算辅助

## 薪资参考范围

**应届 10–18k/月；CRO/生物医药统计师 14–25k/月；顶尖金融量化方向研究生 30k+/月**

## 就业市场备注

医药CRO行业快速扩张，生物统计师需求旺盛。政府统计局体制内是稳定方向。金融/量化是高薪但高门槛路
