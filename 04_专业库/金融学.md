# 专业定位

**金融学**（代码：020301K）是经济学门类的核心专业，学制4年，授予学士学位。

# 核心课程

- 公司金融
- 投资学
- 金融市场学
- 固定收益证券
- 金融衍生品
- 国际金融
- 财务报表分析

# 适合什么学生

- 对本领域有浓厚兴趣，能主动自学
- 逻辑思维严谨，能接受抽象概念训练
- 能接受该专业学制要求的学习节奏与强度

# 不适合什么学生

- 对数字/会计/报表分析极度排斥
- 只想"做金融感觉体面"但不愿深度学习


# 专业优点

- 如果学校平台够强，能摸到银行、券商、基金、资管、投研这类门槛更高的方向。
- 就算不冲头部，银行、保险、风控、财务分析、经营分析这些路径也还算现实。
- 对数字敏感、愿意做长期积累的人，金融学不是不能走，但得认清平台差异。

# 专业缺点

- 行业内部分化非常大，学校层次一拉开，岗位起点就不是一个量级。
- 很多所谓“金融岗”其实是销售、客服、渠道或基础运营，不是你想象里的投研。
- 如果只把它当成“听起来体面”，不补数学、英语、实习和平台，最后容易落到普通赛道。

# 未来前景

- 金融不会消失，但它早就不是“学了就高薪”的时代了。
- 更现实的趋势是分层：顶尖平台的人继续往高阶金融走，普通院校更多还是银行、保险、风控、财务相关路线。
- 真要往上走，学校平台、城市资源、实习和研究能力，缺一个都不轻松。


# 适合的家庭类型

- 家里能接受孩子前期先投入、后期再看回报的家庭。
- 不介意孩子先从银行、券商、保险、风控这类岗位往上爬的家庭。
- 能接受“平台差一点，路就会差很多”这个现实的家庭。

# 发展路线

- 本科阶段先把数学、英语、统计思维和Excel/Python基础打扎实。
- 如果目标是券商、基金、投研、资管，学校平台和实习经历要尽早补。
- 如果目标更稳，银行、保险、风控、财务分析、经营分析这些方向更现实。

# 张雪峰风格点评

金融和财经类专业最大的问题，不是学不会，而是太容易被“看上去很高级”这件事误导。学校平台、城市资源、证书和实习，缺一个都可能把路走窄。它不是不能报，而是你得先想清楚自己是要冲更高的平台，还是接受一条更稳但上限没那么夸张的路。

# 就业方向与薪资参考

> [!NOTE]
> 薪资数据基于行业共识与公开调研，实际因地区、学校层次、个人能力差异显著。

## 主要就业方向

- 投资银行（IBD）分析师
- 私募股权（PE）投资经理
- 证券研究员
- 商业银行对公业务
- 资产管理基金公司

## 薪资参考范围

**国内顶尖投行（中金/中信）应届 20–40k/月；外资IBD应届 30–60k/月；商业银行对公 12–20k/月**

## 就业市场备注

头部岗极为激烈，但行业上限不封顶。PE/VC的carry分红可以远超工资。城市选择（北京/上海）是关键变量
