# 专业定位

**集成电路设计与集成系统**（代码：080710T）是工学门类的核心专业，学制4年，授予学士学位。

# 核心课程

- 模拟集成电路设计
- 数字集成电路设计（Verilog/SystemVerilog）
- 半导体器件物理
- EDA工具与流程
- 芯片验证（UVM）
- VLSI设计方法学
- 封装测试

# 适合什么学生

- 对本领域有浓厚兴趣，能主动自学
- 逻辑思维严谨，能接受抽象概念训练
- 能接受该专业学制要求的学习节奏与强度

# 不适合什么学生

- 数学/电路基础薄弱
- 喜欢快速开发迭代而非深度工程打磨


# 专业优点

- 行业应用面广，和数字化、智能化、自动化升级关系紧密。
- 本科阶段就能建立较明确的技术能力，普通家庭也有机会靠能力吃饭。
- 如果学校平台、项目经历和实习跟得上，上升空间通常比较可观。

# 专业缺点

- 学习门槛不低，课程硬，很多内容不能只靠考前突击。
- 行业更新快，不主动学新工具和新方向，掉队会很明显。
- 头部岗位竞争激烈，平台差异和个人能力差异会被放大。

# 未来前景

- 从当前产业升级方向看，这类专业整体仍属于更值得重点关注的方向。
- 但前景不是“专业名字自动兑现”，而是更看学校平台、项目经历、实习质量和城市资源。
- 如果想走得更好，建议尽早补齐代码、项目、竞赛或工程实践，不要只停留在课程层面。


# 适合的家庭类型

- 普通工薪家庭、愿意靠能力和项目说话的家庭。
- 能接受孩子大学期间持续自学、做项目、卷实习的家庭。
- 如果家庭更看重长期回报和技术含金量，这一类通常更容易讲得通。

# 发展路线

- 本科阶段先把编程基础、数据结构、项目经验打牢。
- 如果学校平台一般，就尽量用竞赛、实习和考研补平台。
- 想冲高薪岗，就往算法、系统、AI工程、核心开发这些方向靠；想稳一点，就走软件开发、测试、运维、数据应用。

# 张雪峰风格点评

这类专业最核心的不是“听起来高科技”，而是你愿不愿意真下功夫。它们大多跟数学、编程、实验、工程能力、项目实践绑得很紧，适合能吃得下硬课、能接受长期积累的人。家里如果更看重稳就业和真实技能，这类方向通常比空泛概念型专业更靠谱，但前提是学校平台和个人能力不能太差。

# 就业方向与薪资参考

> [!NOTE]
> 薪资数据基于行业共识与公开调研，实际因地区、学校层次、个人能力差异显著。

## 主要就业方向

- 芯片后端设计工程师（PR/Layout）
- 模拟IC设计工程师

## 薪资参考范围

**本科应届 15–25k/月；研究生 25–50k/月；顶尖IC设计岗（高通/MTK/海思）硕博 50k+/月**

## 就业市场备注

当前全工科最高薪赛道之一。国家半导体自主可控战略全面提速，IC设计人才严重稀缺，薪资持续高速增长
